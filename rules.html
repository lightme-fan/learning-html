Rules

There are a lot of rules in html.

These are some of the rules.

  1. Tags must be inside of angle brackets, as in <body>
  2. There is a slash in end tags. Example: <h1>Hello</h1>
  3. Mosr tags are in pairs, as in <body></body>
  4. An open element must be closed.
  5. There are some tags without end tags.
  6. All elements must be written in lowercasse.
