All elements

HTML consists hundred elements.

These are the elements in HTML

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The(<abbr>) element is used along with a title attribute to associate a full-text explanation with an abbreviation or acronym.

<article>

The HTML (<article>) element represents a self-contained composition in a document, page, application, or site.

<b>

The HTML (<b>) element is used to bold a word.

<body>

The (<body>) element represents the main content of the document.

<br>

The HTML (<br>) element represents a line break.

<button>

The HTML (<button>) element is used to activate a button.

<caption>

The (<caption>) element represents the title of the table that is its parent, if it has a parent and that is a table element.

<cite>

The (<cite>) element represents the title of a work .

<code>

The (<code>) element represents a fragment of computer code.

<col>

The (<col>) element represents columns.

<colgroup>

The (<colgroup>) element represents a group of one or more columns in the table.

<datalist>

The (<datalist>) element is a set of option elements.

<del>

The (<del>) element is used to remove a word in document.

<div>

The (<div>)element has no meaning at all.

<DOCOTYPE>

The (<DOCOTYPE>)element must be at the beginning of html.

<dd>

The (<dd>)element represents the describtion fo an item.

<dfn>

The (<dfn>)element represents a definiton of a word.

<dt>

The (<dt>)element represents the item which is going to be definited.

<em>

The (<em>)element is used to emphasize.

<fieldset>

The (<fieldset>)element represents a new context.

<figcaption>

The (<figcaption>) element represents a caption.

<footer>

The (<footer>)element represents the end of the section.

<form>

The (<form>)element is used to form a value.

<h1-h6>

The (<h1-h6>)element represents the headings and subheadings.

<head>

The (<head>)element represents the intrduction of the document.

<header>

The element (<header>) is used to represent the headings.

<hr>

The (<hr>) element represents the separation of different topics.

<html>

The (<html>) element represents the whole document.

<i>

The (<i>) element is used to make italics.

<iframe>

The (<iframe>) element represents a nested list.

<img>

The (<img>) element represents an image.

<input>

The (<input>)element is used to edit typed data.

<ins>

The (<ins>)element is used to add a word to a document.

<kbd>

The (<kbd>)element is used to contain a samp element.

<label>

The (<label>)element represents a caption.

<lang>

The(<lang>) element represents the language we use in the document.

<legend>

The (<legend>)element represents a caption after label.

<li>

The (<li>) element represents a list item.

<link>

The (<link>) element links the document to another resources.

<main>

The (<main>) element represents the main of the document.

<mark>

The (<mark>)element is used to highlight words.

<meta>

The (<meta>) element represents the metadata.

<nav>

The (<nav>) element represents the navigation for document which contains the links.

<ol>

The (<ol>) element represents an ordered list.

<optgroup>

The (<optgroup>)element is used to group options.

<option>

The (<option>)element represents an option .

<p>

The (<p>) element represents paragraphs.

<s>

The(<s>) element represents a content which is no accurate anymore.

<small>

The (<small>)element is used to comment.

<section>

The (<section>) element is grouping its content.

<select>

The (<select>)element is used to select an option.

<span>

The (<span>)element represents nothing, but it is useful with global attributes.

<strong>

The (<strong>)element represents the importance of its contents.

<style>

The (<style>) element is used te make styles in a document.

<sub>

The(<sub>) element represents a subscript. It can be used inside a var element.

<summary>

The (<summary>) element represents a summary.

<sup>

The(<sup>) element represents a superscript .

<table>

The (<table>) element represents a table with its dimension.

<tbody>

The (<tbody>) element represents the rows in a table.

<td>

The (<td>) element represents data in a table.

<textarea>

The (<textarea>) element is used to control the element's raw value .

<th>

The (<th>) element represents the headers of a table.

<thead>

The (<thead>) element represents the headers of columns.

<time>

The (<time>)element is used to precise date .

<title>

The (<title>) element represents the document's name.

<tr>

The (<tr>) element represents the rows of cell in a table.

<ul>

The (<ul>) element represents a list of item.

<var>

The (<var>) element represents a variable. This could be a variable in a context.
