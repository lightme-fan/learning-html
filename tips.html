Tips of how to write HTML properly

This page is made to give you some tips in order to help you writing HTML properly.

Writing HTML faster

  • Write a name of a tag, then press TAB in your laptop. It will create an opening and nan ending tag that you wrote.
  • You had better use keyboards shortcut, as in CTRL+P

The tools you should use

There are a lot of tools which can help you fix the mistakes that you might make, but they can't write your code for you. These are some of them:

  • Editorconfig
  • W3C Markup validation service
  • Visual studio
  • Github

Making your works high-quality

  • Indent your code properly.
  • Use Editorconfig in visual studio.
  • Checking your code in W3c Markup Validation service hepls you to know if you have made mistakes or not.
  • Use View word wrap in visual studio.
  • Look at your work by using preview CTRL+k+v.
