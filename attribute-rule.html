Attribute rule

Start tags must be inside of less and greater than characters, and there is a slash in end tags.

HTML attributes are used to modify an element type. They are special words used inside of opening tags. Attributtes support most the elements.

An HTML element usually consists these attributes:

  • Type: is used to specify the content of the element.
  • Name: the purpose of the name attribute is to describe the name for an element.
  • Value: the value attribute is used to specifie the value of an <input> element.
  • Id: when you use a for attribute, you have to use id. The id attribute is used to specify a unique id for an HTML element.

Examples

  • <label for="username">Username</label>
    <input type="submit" id="username" name="username" required>
  • <input type="submit" name="login" value="login">
