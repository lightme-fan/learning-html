Nested rules

All tags in HTML had better be nested correctly. That means when a tag is open, there must a close tag.

When you write a tag inside of an another tag, you need to be careful to the order where each tag should be open and ended.

These are some examples:

  1. Right nested :
    <body> <h1> <p> </p> </h1> </body>
  2. Wrong nested :
    <nav> <ul> <li> </ul> </nav> </li>
